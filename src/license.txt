forceASPI Installer - made with NSIS

    This installer will install Adaptec ASPI32 4.60 (1021) layer. Before changing anything on the system it will backup any current installation of ASPI and hence provide an easy way to return to the old settings in case anything goes wrong.
    Originally these set of scripts were written by Wimpy @ Flexion.org.
